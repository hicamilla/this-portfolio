What I built (short)

A responsive weather app built with HTML, CSS, and JavaScript that lets users search for any city and view current conditions, temperature, wind, and humidity, plus a 5-day forecast. Created to practice API integration and mobile-friendly UI development.

Key bits

  • Real-time weather data via city search.
  • Form input with validation and error handling.
  • 5-day forecast with visual icons.
  • Responsive layout with light/dark mode toggle.
  • Dynamic temperature display and weather icons.

Project Facts

  • Role: Front-end Developer & Content Creator
  • Time: 1 week
  • Scope: Built during the SheCodes Workshop powered by curiosity and documentaries bingeing.
