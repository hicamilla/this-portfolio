What I built (short)

A responsive world clock app built with HTML, CSS, and JavaScript that shows the current time across major cities. Users can select a location from a dropdown to view its local time in real-time. Created to practice time zone handling, JavaScript's Date object, and building interactive, user-friendly UI components.

Key bits

  • Live clock with automatic real-time updates.
  • Dropdown menu to select global time zones.
  • Styled with Flexbox and CSS variables.
  • Responsive layout with clean, semantic structure.
